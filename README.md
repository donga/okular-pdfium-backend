okular-pdfium-backend
====================

This is a PDF backend for Okular using PDFium.

Requirements
------------
- cmake
- Okular devel
- PDFium library

Compilation
-----------
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```
